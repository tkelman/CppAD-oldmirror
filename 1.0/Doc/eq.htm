AD Assignment Operator

Syntax
y = x

Purpose
Assigns a new value to the AD object y. The object y will have the same dependence on the independent variables as x (y is a variable if and only if x is a variable).

x
The operand x has one of the following prototypes
     const int                    &x
     const Base                   &x
     const AD<Base>               &x
     const VecAD<Base>::reference &x

y
This operation is a member function for y which has the one of the following prototypes
     AD<Base>               y
     VecAD<Base>::reference y

Result

AD<Base>
If y has type AD<Base>, the result of this assignment can be used as a reference to y. For example, if z has the following type
     AD<Base> z
then the syntax
     z = y = x
will assign the value of x to z (as well as to y).

VecAD<Base>::reference
If y has type VecAD<Base>::reference, this assignment has a void result. For example, the following syntax is not valid:
     z = y = x

Operation Sequence
This is an AD of Base atomic operation and hence is part of the current AD of Base operation sequence .

Example
The file Eq.cpp contains an example and test of these operations. It returns true if it succeeds and false otherwise.
