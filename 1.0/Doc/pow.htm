The AD Power Function

Syntax
z = pow(x, y)

Purpose
Determines the value of the power function which is defined by  \[
     {\rm pow} (x, y) = x^y
\]  .

x
The argument x has one of the following prototypes
     const AD<Base>               &x
     const VecAD<Base>::reference &x

y
The argument y has one of the following prototypes
     const AD<Base>               &x
     const VecAD<Base>::reference &x
(see PowInt for the case where y is an int).

z
The result z has prototype
     AD<Base> z

Standard Types
A definition for the pow function is included in the CppAD namespace for the case where both x and y have the same type and that type is float, double, std::complex<float>, or std::complex<double>. For these cases, the result has the same type as the arguments.

Operation Sequence
The AD of Base operation sequence used to calculate z is independent of x and y.

Example
The file Pow.cpp contains an example and test of this function. It returns true if it succeeds and false otherwise.
