An Example Exponential Approximation

Syntax
# include "ExpApx.hpp"
y = ExpApx(x, e)

Purpose
This is an example algorithm that is used to demonstrate Algorithmic Differentiation.

Mathematical Form
The exponential function can be defined by  \[
     \exp (x) = 1 + x^1 / 1 ! + x^2 / 2 ! + \cdots 
\]  We define  k ( \varepsilon )   as the smallest non-negative integer such that  \varepsilon \geq x^k / k ! ; i.e.,  \[
k( \varepsilon ) = \min \{ k \in {\rm Z}_+ \; | \; \varepsilon \geq x^k / k ! \}
\]  The mathematical form for our approximation of the exponential function is  \[
\begin{array}{rcl}
{\rm ExpApx} (x , \varepsilon ) & = & \left\{
\begin{array}{ll}
\frac{1}{ {\rm ExpApx} (-x , \varepsilon ) } 
     & {\rm if} \; x < 0 
\\
1 + x^1 / 1 ! + \cdots + x^{k( \varepsilon)} / k( \varepsilon ) !
     & {\rm otherwise}
\end{array}
\right.
\end{array}
\] 

include
The include command in the syntax is relative to
     omhelp-yy-mm-dd/Introduction
where omhelp-yy-mm-dd is the distribution directory created during the extraction step of a Unix or Window installation of CppAD.

x
The argument x has prototype
     const Type &x
(see Type below). It specifies the point at which to evaluate the approximation for the exponential function.

e
The argument e has prototype
     const Type &e
It specifies the accuracy with which to approximate the exponential function value; i.e., it is the value of  \varepsilon in the exponential function approximation defined above.

y
The result y has prototype
     Type y
It is the value of the exponential function approximation defined above.
