Routines That Speed Test CppAD

Purpose
CppAD has a set of speed tests that are used to determine if certain changes improve its execution speed. This section explains how you can run these tests on your computer.

Unix
In a unix operating system (or in a Cygwin shell on an Microsoft system), you must first configure CppAD to run its speed tests. To do this, run the install configure command with the --with-Speed command line option included. You can then compile link and run the speed tests with the following commands (relative to the distribution directory):
     cd Speed
     make 
     ./Speed
Windows
In Microsoft Visual Studio open the project file Speed/Speed.sln (relative to the distribution directory). Then set the configuration to the Release version. Then select Build | Speed.
