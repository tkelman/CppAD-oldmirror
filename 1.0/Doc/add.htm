AD Binary Addition Operator

Syntax
z = x + y

Purpose
Computes the sum of x plus y where one of the two operands has type AD<Base> or VecAD<Base>::reference.

x
The operand x has one of the following prototypes
     const int                    &x
     const Base                   &x
     const AD<Base>               &x
     const VecAD<Base>::reference &x

y
The operand y has one of the following prototypes
     const int                    &y
     const Base                   &y
     const AD<Base>               &y
     const VecAD<Base>::reference &y

z
The result z has type
     const AD<Base> z

Operation Sequence
This is an AD of Base atomic operation and hence is part of the current AD of Base operation sequence .

Derivative
If  f and  g are Base functions  \[
     \D{[ f(x) + g(x) ]}{x} = \D{f(x)}{x} + \D{g(x)}{x}
\] 

Example
The file Add.cpp contains an example and test of these operations. It returns true if it succeeds and false otherwise.
