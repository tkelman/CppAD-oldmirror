Examples

Introduction
This section organizes the information related to the CppAD examples. Each CppAD operation has its own specific example, for example Add.cpp is an example for Add . Some of the examples are of a more general nature (not connected of a specific feature of CppAD). In addition, there are some utility functions that are used by the examples.

Running Examples
The installation instructions show how the examples can be run on your system.

Choosing CppADvector
Many of the examples use the CppADvector preprocessor symbol to determine which SimpleVector template class is used with the examples.
