The Integer Power Function

Syntax
# include <CppAD/PowInt.h>
y = pow(x, n)

Purpose
Determines the value of the power function  \[
     {\rm pow} (x, n) = x^n
\]  for integer exponents n.

Include
The file CppAD/PowInt.h is included by CppAD/CppAD.h but it can also be included separately with out the rest of the CppAD routines. Including this file defines pow within the CppAD namespace.

x
The argument x has prototype
     const Type &x

n
The argument n has prototype
     int n
(see pow for the case where n is an AD<Base> object).

y
The result y has prototype
     Type y

Type
The type Type must support the following operations where a and b are Type objects and i is an int:
Operation    Description Result Type
Type a(i) construction of a Type object from an int Type
a * b binary multiplication of Type objects Type
a / b binary division of Type objects Type

Operation Sequence
The Type operation sequence used to calculate y is independent of x.

Example
The file PowInt.cpp contains an example and test of this function. It returns true if it succeeds and false otherwise.
